## Testing Layers
- Unit: Vitest + Testing Library
- Integration: Playwright
- E2E: Cypress
- Visual: Percy or Chromatic
- Performance: Lighthouse CI
- Accessibility: axe-core
- Cross-browser: BrowserStack
